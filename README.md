# Todo-List-Project
“Developed a Python-based desktop Todo application with SQLite database, task management, filtering, and real-time statistics."
# 📝 Advanced Todo List Application (Python GUI)

## 📌 Project Description

This is a fully functional Todo List desktop application built using Python with a modern GUI.
The app allows users to manage tasks efficiently with features like priority levels, deadlines, search, filters, and real-time statistics.

---

## 🚀 Features

### 🧾 Task Management

* Add new tasks with title, description, priority, and deadline
* Edit existing tasks
* Delete tasks
* Mark tasks as **Done / Pending**

---

### 🎯 Priority System

* Multiple priority levels:

  * Low
  * Medium
  * High
  * Critical
* Color-coded priority indicators for better visualization

---

### ⏰ Deadline & Notifications

* Set deadlines for tasks
* Overdue task detection
* Notification for pending overdue tasks

---

### 🔍 Search & Filters

* Real-time search functionality
* Filter tasks by:

  * Status (Pending / Done)
  * Priority (Low to Critical)

---

### 📊 Dashboard & Stats

* Total tasks count
* Pending tasks count
* Completed tasks count
* Critical tasks tracking

---

### 💾 Database Integration

* Uses SQLite database for data storage
* Persistent task saving (data saved even after closing app)

---

### 🎨 User Interface

* Modern GUI using CustomTkinter
* Dark mode design
* Responsive and user-friendly layout
* Scrollable task list

---

## 🛠️ Technologies Used

* Python
* SQLite3
* CustomTkinter (GUI Framework)

---

## ▶️ How to Run

1. Install required library:

```bash
pip install customtkinter
```

2. Run the application:

```bash
python your_file_name.py
```
